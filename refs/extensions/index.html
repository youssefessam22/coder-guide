VScode:

Code Navigation and Productivity:

  • Bracket Pair Colorizer: Helps to identify matching brackets with colors.
  • Auto Close Tag: Automatically closes HTML/XML tags.
  • Auto Rename Tag: Renames paired HTML/XML tags.
  • Path Intellisense: Autocompletes filenames in your code.
  • Code Spell Checker: Checks for spelling errors in your code comments and strings.

Version Control:

  • GitLens: Provides powerful Git capabilities inside VSCode.
  • GitHub Pull Requests and Issues: Allows you to manage your pull requests and issues directly from VSCode.

Syntax Highlighting and Formatting:

  • Prettier: Automatically formats your code to a consistent style.
  • ESLint: Integrates ESLint into VSCode for JavaScript/TypeScript linting.
  • Rainbow Brackets: Adds color to brackets to improve code readability.
  • Color Highlight: Highlights colors found in your code.

Language Support:

  • Python: Offers support for Python development, including linting and debugging.
  • JavaScript (ES6) code snippets: Provides snippets for JavaScript (ES6) syntax.
  • Java Extension Pack: Brings essential Java tools to VSCode.
  • C# for Visual Studio Code: Provides support for C# development.

Debugging

  • Debugger for Chrome: Allows you to debug JavaScript code running in the Chrome browser.
  • Python Extension Pack: Includes necessary tools for Python development, including debugging.

Themes and UI Enhancements:

  • Material Theme: Applies the popular Material Design theme to VSCode.
  • Material Icon Theme: Provides Material Design icons for files and folders.
  • Custom CSS and JS Loader: Allows you to customize VSCode's UI using CSS and JavaScript.

Testing:

  • Jest: Provides Jest integration for VSCode for JavaScript testing.

Other Useful Extensions:

  • Live Server: Launches a local development server with live reload feature for web development.
  • Remote - SSH: Allows you to work with remote machines over SSH.
  • Docker: Adds syntax highlighting, commands, and more for Docker files and Docker Compose files.

Browser

Privacy and Security:

  • uBlock Origin: Blocks ads, trackers, and malware domains for faster and safer browsing.
  • HTTPS Everywhere: Ensures secure connections to websites by redirecting to HTTPS.
  • Privacy Badger: Blocks invisible trackers and helps you maintain privacy while browsing.
  • LastPass: Password manager that securely stores your passwords and autofills them when needed.

Productivity:

  • Grammarly: Checks spelling and grammar mistakes across the web, including emails, social media, and more.
  • Pocket: Saves articles, videos, and web pages to view later across any device.
  • Evernote Web Clipper: Clips web pages directly into your Evernote account for later reference.
  • Tab Snooze: Allows you to snooze tabs and set them to reopen at a later time.

Developer Tools:

  • JSON Formatter: Formats JSON responses in a more readable way.
  • Wappalyzer: Identifies technologies used on websites, including CMS, frameworks, and more.
  • ColorZilla: Allows you to pick colors from web pages and create color palettes.
  • Octotree: Provides a code tree for GitHub repositories, making it easier to navigate codebases.

Content Enhancement:

  • The Great Suspender: Suspends inactive tabs to reduce memory usage and improve browser performance.
  • Dark Reader: Inverts website colors to reduce eye strain and improve readability.
  • Mercury Reader: Removes clutter from web pages, leaving only the main content for distraction-free reading.

Social Media and Communication:

  • Hootsuite Hootlet: Share web content directly to your social media accounts.
  • Bitmoji: Allows you to easily insert Bitmoji avatars into web forms and social media posts.
  • Checker Plus for Gmail: Provides desktop notifications and quick access to Gmail.

Search and Navigation:

  • Google Translate: Translates web pages and selected text into different languages.
  • SearchPreview: Displays website thumbnails next to search results to give you a preview of the content.
  • The Camelizer: Shows historical pricing data for products on Amazon to help you make informed purchasing decisions.
